#Fichiers mécaniques

Le fichier global.xml s'ouvre avec l'utilitaire umlet ou la version en ligne umletino.

##Kicad

Les schémas électroniques sont conçus avec le logiciel open source Kicad.


| Fichiers/dossiers         | Explication                                             |
|---------------------------|---------------------------------------------------------|
| robair.lib                | librairie kicad des composants du projet                |
| robair.pretty             | librairie d'empreinte kicad des composants du projet    | 
| alim5v                    | le projet de la carte d'alimnetation                    | 
| cartehaute                | le projet de la carte de connexion placé dans le buste  |
